Richieste HTTP

come fuzniona il we?

client (utente, browser) --> richiesta http --> server (node/express)

server (node/express) --> risposta http --> client (utente, browser)


Metodi HTTP

    - GET : leggere dati
    - POST : inviare dati
    - PUT : modificare dati
    - DELETE : eliminare dati

    GET : 
        -sito.it/api/ordini (vedere tutti gli ordini) 
        -sito.it/api/ordini/:id (vedere un ordine specifico)
        
    POST :
        -sito.it/api/ordini (eseguire un ordine)

    PUT : 
        -sito.it/api/ordini/:id (modificare un ordine)
    
    DELETE :
        -sito.it/api/ordini/:id (eliminare dati)
